# 📚 Library Management System

A full-stack Library Management System built using **Node.js, Express, MySQL, HTML, CSS, and JavaScript**.

---

## 🚀 Features

### 👨‍💼 Admin
- Add / Update / Delete books  
- Issue books to users  
- Return books  
- View reports  

### 👨‍🎓 Student
- View all books  
- Search books  
- Check issue status  

---

## 🛠️ Tech Stack
- Frontend: HTML, CSS, JavaScript  
- Backend: Node.js, Express.js  
- Database: MySQL
